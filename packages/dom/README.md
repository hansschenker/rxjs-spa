# @rxjs-spa/dom

DOM adapter layer for rxjs-spa.

- sources: `events`, `textChanges`, `attrChanges`, `classChanges`, `hasClass`, `valueChanges`, `checkedChanges`
- sinks: `text`, `html`, `attr`, `prop`, `style`, `classToggle`, `dispatch`, `mount`
- lists: `renderList`, `renderKeyedList`, `renderKeyedComponents`
